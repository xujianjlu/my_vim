## my_vim
================
This is my vim configure, Include vimrc, and the plugins.

Having fun !

## Usage
================
### 1
backup your own '.vimrc' and the folder '.vim/' from your 'home' folder.

### 2
::
mkdir my_vim
cd my_vim
git clone git@github.com:xujianjlu/my_vim.git
cp .vimrc ~/.vimrc
cp -r .vim/ ~/.vim
